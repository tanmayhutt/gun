# GUN - The Official Rulebook

Welcome to the official repository for **GUN**, a high-stakes, 6-player card game. This project hosts the beautifully designed, interactive web-based rulebook.

## 🃏 About The Game
**GUN** is played with two standard 52-card decks and focuses on strategic discarding, rapid gameplay, and the thrill of avoiding 100 penalty points. The game combines elements of matching, sequential play, and special action cards to create a highly dynamic and engaging experience.

## ✨ Features
- **Complete Rulebook:** Detailed, easy-to-read instructions covering setup, gameplay, card powers, and scoring.
- **Interactive UI:** Built with a modern, glassmorphism-inspired design system.
- **Custom CSS Cards:** Fully functional, CSS-drawn playing cards with 3D flip animations and dynamic scaling.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Zero Dependencies:** Pure HTML and CSS. No JavaScript frameworks required.

## 🚀 Deployment
This project is built as a static site and is pre-configured to deploy automatically via GitHub Actions to GitHub Pages. Any changes pushed to the `main` branch will instantly update the live site.

## 🛠️ Local Development
To view or modify the rulebook locally:
1. Clone the repository: `git clone https://github.com/tanmayhutt/gun.git`
2. Open `index.html` in your web browser.
3. Edit `index.html` and `style.css` to make changes.
